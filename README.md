# Task Management Web App

A web-based task tracking system developed using Flask and SQLite to replace manual notebook logging in logistics operations.

## Features
- Create / delete / complete tasks
- Group tasks by assignee
- Excel logging for performance tracking
- Category-based task management

## Tech Stack
- Python (Flask)
- SQLite
- OpenPyXL
- HTML + CSS

## Impact
Reduced operational backlog from 39.25% to 16.82%.
